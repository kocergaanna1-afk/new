# Интеграция TradingView и Finam (Природный Газ)

Этот проект реализует веб-сервер для автоматической торговли фьючерсами на природный газ (Natural Gas) на Московской Бирже через брокера Финам по сигналам из TradingView.

## Компоненты

1.  **FastAPI Webhook Server (`main.py`)**: Принимает JSON сигналы от TradingView.
2.  **Finam Client (`finam_client.py`)**: Общается с API Финама для размещения ордеров.
3.  **Config**: Настройки через переменные окружения.

## Предварительные требования

1.  Аккаунт TradingView (Pro/Pro+/Premium для использования Webhooks).
2.  Торговый счет в Финам.
3.  Токен доступа API Финам (Trade API).

## Установка

1.  Клонируйте репозиторий.
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
3.  Создайте файл `.env` на основе `.env.example` и заполните ваши данные:
    ```ini
    FINAM_API_TOKEN=ваш_токен
    FINAM_CLIENT_ID=ваш_клиентский_id
    TRADING_ACCOUNT_ID=ваш_счет
    BOARD=FUT
    DEFAULT_TICKER=NGH6
    ```
    *Примечание: `NGH6` - это пример тикера фьючерса на газ (март 2026). Тикеры меняются каждый месяц/квартал. Проверяйте актуальный тикер в терминале Финам.*

## Запуск

Запустите сервер:

```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```

Сервер будет доступен по адресу `http://ВАШ_IP:8000`.

## Настройка TradingView

1.  Откройте график Natural Gas (например, `NG1!` или `MOEX:NGH6`).
2.  Создайте новый Alert (Оповещение).
3.  В настройках оповещения выберите **Webhook URL**.
4.  Вставьте URL вашего сервера: `http://ВАШ_ПУБЛИЧНЫЙ_IP:8000/webhook`.
    *Примечание: Для локального тестирования используйте ngrok: `ngrok http 8000`.*
5.  В поле **Message** вставьте JSON payload:

**Для покупки:**
```json
{
    "ticker": "NGH6",
    "action": "buy",
    "quantity": 1,
    "passphrase": "secret_if_enabled"
}
```

**Для продажи:**
```json
{
    "ticker": "NGH6",
    "action": "sell",
    "quantity": 1,
    "passphrase": "secret_if_enabled"
}
```

## Важные предупреждения

- **Тестирование**: Всегда тестируйте на демо-счете или с минимальным объемом (1 лот).
- **Тикеры**: Фьючерсы имеют дату экспирации. Не забывайте менять тикер в `.env` или в сообщении TradingView при смене контракта.
- **Безопасность**: Используйте HTTPS и добавьте проверку `passphrase` в `main.py` для защиты от несанкционированных запросов.
- **Риски**: Автоматическая торговля сопряжена с риском потери средств. Автор кода не несет ответственности за убытки.
