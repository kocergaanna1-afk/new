# Интеграция TradingView и Finam (Природный Газ)

Этот проект реализует автоматизированную торговлю фьючерсами на природный газ (Natural Gas) на Мосбирже через Финам по сигналам TradingView. 
**Поддерживает автоматический ролловер (смену контрактов) и восстановление позиции.**

## Особенности
- **Авто-ролловер**: Система сама определяет актуальный фьючерс (например, NGH6 -> NGJ6) на основе текущей даты.
- **Восстановление позиции**: Если позиция была закрыта биржей (экспирация) или сбой, но сигнала на закрытие не было, бот восстановит позицию на новом актуальном контракте.
- **Состояние**: Хранит желаемую позицию (лонг/шорт) в локальной базе данных (`bot_state.db`).

## Компоненты
- `main.py`: Веб-сервер для Webhook.
- `background.py`: Фоновый процесс проверки позиций и смены контрактов.
- `ticker_utils.py`: Логика расчета тикеров (коды месяцев F, G, H...).
- `database.py`: SQLite база для хранения состояния.
- `finam_client.py`: Работа с API Финама.

## Настройка
1.  Установите зависимости: `pip install -r requirements.txt`
2.  Настройте `.env` (см. `.env.example`).
3.  **Важно**: В `ticker_utils.py` можно настроить `ROLLOVER_DAY` (день перехода на новый контракт, по умолчанию 25-е число).

## Запуск
```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```
При запуске создастся БД и запустится фоновая задача проверки.

## Настройка TradingView
Теперь в TradingView можно не указывать тикер явно, бот сам подставит актуальный.

**Payload для покупки (Long):**
```json
{
    "action": "buy",
    "quantity": 1
}
```

**Payload для продажи (Short):**
```json
{
    "action": "sell",
    "quantity": 1
}
```

**Payload для закрытия всех позиций (Flat):**
```json
{
    "action": "flat",
    "quantity": 0
}
```

## Логика Ролловера
Раз в минуту бот:
1. Вычисляет актуальный тикер по дате.
2. Если тикер изменился (наступил новый месяц):
   - Закрывает позицию на старом тикере (если есть).
   - Открывает позицию на новом тикере (если согласно стратегии мы должны быть в позиции).
3. Если позиция исчезла (экспирация), но в БД записано, что мы должны быть в рынке -> открывает заново по рынку.
